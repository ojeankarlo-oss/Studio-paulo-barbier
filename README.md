# Studio Paulo Barbier — App de Agendamento

App web progressivo (PWA) para agendamento de cortes e barba.

## Como publicar no GitHub Pages

1. Crie um repositório chamado `studio-paulo-barbier` no GitHub
2. Faça upload dos arquivos `index.html` e `manifest.json`
3. Vá em Settings → Pages → Source: Deploy from branch → main → / (root)
4. Aguarde 2 minutos e acesse: `ojeankarlo-oss.github.io/studio-paulo-barbier`

## Configuração inicial (Paulo)

1. Acesse o app → clique em ADM
2. Senha padrão: **1234**
3. Vá em Configurações e preencha:
   - Chave PIX
   - Instagram
   - Token e Chat ID do Telegram
   - Nova senha e código de emergência
